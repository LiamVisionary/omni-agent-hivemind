# Simple Website Starter

A clean, responsive one-page website starter built with plain HTML, CSS, and a tiny bit of JavaScript.

## Open it

On macOS:

```sh
open sites/simple-website/index.html
```

Or open this file directly in a browser:

```text
/Users/liam/Documents/code/projects/hivemind-os/sites/simple-website/index.html
```

## Customize it

- Change the site name in the header and `<title>`.
- Update the hero headline and supporting copy.
- Replace the three service cards with your real offers or features.
- Change colors in the `:root` CSS block.
- Replace `hello@example.com` with your real contact link.
